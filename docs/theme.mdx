# Theme Configuration

This documentation site uses Nextra theme with custom configurations for enhanced user experience.

## Overview

The theme is configured in `theme.config.tsx` at the root of the project. This file contains all the theme settings and customizations.

## Dark Mode

The theme supports dark mode with three options:

- **Light**: Light theme
- **Dark**: Dark theme
- **System**: Follows system preference

Users can toggle between themes using the theme switcher in the navigation bar.

## Primary Colors

The primary color scheme is customized with:

- **Primary Hue**: 210 (blue tone)
- **Primary Saturation**: 100% (full saturation)

These values create a professional blue color scheme that matches the HUMANIKA branding.

## Sidebar Configuration

The sidebar is configured with:

- **Auto Collapse**: Enabled for better navigation on smaller screens
- **Toggle**: Users can manually expand/collapse the sidebar

## Navigation

The navigation includes:

- **Logo**: HUMANIKA branding
- **Search**: Full-text search across documentation
- **Theme Switcher**: Toggle between light/dark/system themes
- **GitHub Link**: Direct link to the documentation repository

## Footer

The footer displays:

- **Copyright Text**: HUMANIKA Documentation
- **Links**: Additional navigation links (configurable)

## Repository Integration

The documentation is linked to the GitHub repository at `https://github.com/shuding/nextra/tree/main/docs` for:

- **Edit Links**: Direct links to edit pages on GitHub
- **Source Access**: Easy access to source files

## Customization

To customize the theme, modify the `theme.config.tsx` file:

```tsx
import React from "react";
import { DocsThemeConfig } from "nextra-theme-docs";

const config: DocsThemeConfig = {
  logo: <span>HUMANIKA</span>,
  project: {
    link: "https://github.com/your-repo/docs",
  },
  docsRepositoryBase: "https://github.com/your-repo/docs",
  footer: {
    text: "Your Documentation",
  },
  sidebar: {
    autoCollapse: true,
  },
  themeSwitch: {
    useOptions() {
      return {
        light: "Light",
        dark: "Dark",
        system: "System",
      };
    },
  },
  primaryHue: 210,
  primarySaturation: 100,
};

export default config;
```

## Available Options

### Logo

- Type: `React.ReactNode`
- Default: `<span>HUMANIKA</span>`

### Project Link

- Type: `object`
- Properties: `link` (string)

### Repository Base

- Type: `string`
- Used for edit links

### Footer

- Type: `object`
- Properties: `text` (string)

### Sidebar

- Type: `object`
- Properties: `autoCollapse` (boolean)

### Theme Switch

- Type: `object`
- Customizable theme options

### Primary Colors

- `primaryHue`: Number (0-360)
- `primarySaturation`: Number (0-100)
